## install

```
npm i lottie-mini
```

## use

```
import { miniLottie } from 'lottie-mini';
const miniData = miniLottie(data, config);
```

## config

| key      | type     | default                     | desc             |
| -------- | -------- | --------------------------- | ---------------- |
| copy     | boolean  | false                       | copy source data |
| dropList | string[] | ['nm', 'tyName', 'n', 'mn'] | dropKeyList      |
